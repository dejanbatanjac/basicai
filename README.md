# Project Title

BasicAI

## Description

Set of Python classes for solving basic AI problems.

## Getting Started

### Dependencies

* PyTorch 1.0+
* Numpy
* Matplotlib

### Installing

`pip install basicai`
or
`pip install git+https://github.com/dejanbatanjac/basicai.git`

<sub>To uninstall:  `pip uninstall basicai`</sub>
